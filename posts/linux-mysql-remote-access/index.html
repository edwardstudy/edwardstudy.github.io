Linux下设置Mysql远程连接

2015-03-05
1 min read
MySQL , Linux

最近使用mysql比较多，常用的功能记录一下

1.首先加入mysql命令行

2.输入权限命令

mysql> G(BAE和谐)RANT ALL PRIVILEGES on *.* to username@'IP address' identified by 'password';

这条命令的意思依次为

G(BAE和谐)RANT ALL PRIVILEGES 授予所有权限

*.* 依次为数据库名和表名，*.* 表示所有表。username和password为验证信息。IP address为访问者IP，%为不限定地址。

就是这么简单，以后遇到命令也会记录下来，我这脑子。
